<h2>🛠️ run project locally and development mode</h2>
<p>☝️ first of all</p>

```
git clone https://github.com/MohammadBohluli/react-todo-app
```

<p>👉 and </p>

```
cd react-todo-app
```

<p>👆 then </p>

```
npm install
npm run dev
```
